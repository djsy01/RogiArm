# RogiArm 프로젝트

## 개요
RogiArm은 학교 캡스톤디자인(졸업 작품)으로 제작 중인 로봇 팔 제어 관련 프로젝트입니다. 현재는 프론트엔드 React 기반의 UI 설계를 진행 중이며, 실제 로봇 제어나 통신, 백엔드 연동은 아직 구현되지 않은 상태입니다.

## 사용 기술
- React
- JavaScript
- HTML/CSS

## 주요 기능
- **UI 초기 구성**: 로봇 팔 제어를 위한 버튼, 슬라이더 등 제어 요소의 레이아웃 설계.
- **페이지 구성**: 프로젝트 소개, 로봇 팔 컨트롤 패널 페이지, 아두이노 설명 페이지 등 여러 페이지로 구성 예정.
- **컴포넌트 설계**: 향후 기능 확장을 고려한 컴포넌트 기반 구조로 제작.

## 기술 포인트
- SPA 구조로 제작, React Router를 이용한 페이지 전환 구성.
- 컨트롤 UI를 직접 설계하여 사용자의 입력 방식(UI/UX)을 실험.
- 향후 아두이노 및 백엔드 연동을 고려한 컴포넌트 명세 및 레이아웃 사전 설계.

## 현재까지의 어려움
- **기획과 설계의 간극**: 아직 하드웨어 및 백엔드 연동이 구현되지 않아 UI 요소가 실제 동작과 연계되지 않는 한계 존재.
- **레이아웃 구조화**: 제어 UI를 직관적으로 배치하기 위한 고민이 많았음. 여러 레이아웃을 실험하며 사용자 입장에서 보기 쉬운 형태로 개선 중.
